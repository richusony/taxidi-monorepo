# 🚗 Taxidi 2.0

**Taxidi 2.0** is a modern car rental platform designed to connect vehicle owners (partners) with customers in a seamless and secure way.

Built with scalability, performance, and real-world traffic handling in mind, Taxidi aims to provide a reliable booking experience for users while offering partners efficient vehicle and earnings management.

The platform includes:

- 🔐 Centralized authentication system
- 👤 Customer application
- 🚘 Partner dashboard
- 🛠️ Admin control panel
- 💳 Secure online payment integration
- ⚡ High-performance backend architecture

Taxidi 2.0 focuses on clean architecture, smooth user experience, and production-grade reliability.
